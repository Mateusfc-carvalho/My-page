Sobre o Site

Este site foi criado especificamente para mostrar uma mistura de Designer & Responsividade Atrativa. Voltado para o Front-end.

Ferramentas Usadas

HTML5

O HTML organiza os blocos de conteúdo — como títulos, parágrafos, imagens e links — e diz ao navegador o que mostrar.

CSS3

O CSS define cores, fontes, tamanhos e posições, criando um layout bonito e responsivo.

Javascript

O JavaScript permite interações, mudanças dinâmicas de conteúdo e animações.

Sass

Sass facilita a escrita e a organização de estilos com recursos avançados, como variáveis.

Bootstrap

Bootstrap fornece componentes pré-construídos para criar interfaces responsivas rapidamente.
