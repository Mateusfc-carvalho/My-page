Sobre o Site

Este site foi criado específicamente para mostrar uma mistura de Designer & Responsividade Atrativa. Voltado para o Front-end.

Linguagens Usadas

HTML5

O HTML(HyperText Markup Language) é como o esqueleto de uma página. Ele organiza os blocos de conteúdo — como títulos, parágrafos, imagens e links — e diz ao navegador o que mostrar.

CSS3

O CSS (Cascading Style Sheets) é o estilista do site. Com ele, definimos cores, fontes, tamanhos, espaçamentos e posições, criando um layout bonito e responsivo.Ele transforma a estrutura crua do HTML em algo visualmente agradável.

Javascript

O JavaScript é o cérebro por trás da ação. Ele permite que o site responda a cliques, mude conteúdo dinamicamente, valide formulários, crie animações,e muito mais. É o que torna a experiência realmente interativa.
